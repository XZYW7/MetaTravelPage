# Thailand Travel Guide Example

This is a complete example demonstrating how to use the travel-page framework to create a Thailand travel guide application.

## Overview

This example shows:
- A complete journey configuration generated by an Agent
- Multiple card types (cover, standard, reference)
- Timeline sections for chronological events
- Info boxes for important notices
- Expandable sections for detailed content
- Phrasebook with Thai language phrases
- Timer feature for activities
- Diary feature for personal records

## Files

- `journey.json` - Main configuration file (Agent-generated)
- `assets/audio/` - Thai language audio files
- `assets/images/` - Image assets

## How to Use

1. Copy the `config/journey.json` file to the root `config/` directory
2. Copy the `assets/` folder to the root directory
3. Open `index.html` in a browser

## Agent Generation

This configuration was generated using the `AGENT_PROMPT.md` guide. The Agent:

1. Analyzed travel materials about Thailand
2. Extracted key information (dates, activities, contacts)
3. Organized content into logical card structure
4. Generated the complete `journey.json` configuration

## Customization

To customize this example:

1. Edit `journey.json` to modify content
2. Replace audio files in `assets/audio/` with your own
3. Add images to `assets/images/`
4. Adjust theme colors in the `app.theme` section

## Features Used

- **Diary**: Enabled for personal journaling
- **Timer**: Enabled with 15/30/45/60 minute presets
- **Phrasebook**: Thai language phrases with audio support

## Schema Compliance

This example fully complies with `schema/journey.schema.json` version 1.0.0.
