# 泰国旅行指南示例

本示例展示如何使用 travel-page 框架创建一个完整的泰国旅行指南应用。

## 示例内容

- **10 张卡片**：封面、行前准备、6 天行程、实用信息
- **泰语短语**：10 个常用短语（带音频发音）
- **实用表格**：预算参考、行程总览、购物推荐
- **提示信息**：着装要求、注意事项等
- **可展开详情**：行李清单、大象营选择等

## 使用方法

### 方式 1：直接使用

本示例的配置已复制到 `config/journey.json`，音频已复制到 `assets/audio/`。直接启动服务器即可查看：

```bash
start-server.bat
```

### 方式 2：从示例恢复

如需恢复到本示例配置：

```bash
# 复制配置
copy examples\thailand-travel\journey.json config\journey.json

# 音频文件在 assets/audio/ 中，无需额外操作
```
